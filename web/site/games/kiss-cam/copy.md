## Funny-face payoff
Ridiculous sustained duck-face / kissy-face on camera for the full 30-second round. Your cheeks puff, brows lift, lips purse — all captured in the corner PiP so every heart you pop is a moment of beautiful humiliation that begs to be screen-recorded and shared.

## Control
mouthPucker blendshape via MediaPipe FaceLandmarker. Pucker threshold: 0.35. Shooting rate: continuous while holding pucker (8-frame cooldown between bursts).

## Mechanics
- 30-second timed round
- Heart targets drift in from left/right edges, bounce off top/bottom
- Pucker fires kiss-hearts upward in a spread pattern
- Pop a target heart → +1 score + particle burst + screen shake
- Timer bar changes color: teal → gold → pink as time runs low
- Dare-shareable score via share.js
