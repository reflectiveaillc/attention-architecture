# Attention Architecture

> Project root: `/Users/manuel/coo/attention-architecture`

A two-part effort to turn the neuroscience of attention into a revenue-generating games factory.

1. **The framework** — a field dossier on how social interfaces hack known neurological circuits to capture and hold human attention.
2. **The factory** — "LOOP", a self-improving pipeline that uses that framework to discover, build, ship, A/B-test, and scale simple games — distributed by AI-generated hook videos on YouTube Shorts, Instagram Reels, and TikTok.

Both halves share one thesis: **you design for a brain before you design for a person.** The framework names the circuits; the factory weaponizes the humane ones and treats the exploitative ones as a known quantity to be used deliberately, not emergently.

## How to run

```bash
cd /Users/manuel/coo/attention-architecture
npm run loop     # one item end-to-end through all 7 stages → run report + verdict
npm run serve    # preview the games site at http://127.0.0.1:4620 (events → state/events/dev.jsonl)
open web/attention-dossier.html     # the framework
open web/loop-control-room.html     # the LOOP control room (interactive — decisions persist locally)
```

`npm run loop` does the whole thing: reads the trend feed (+ past evidence), ideates a concept, prioritizes it, captures a fresh 9:16 hook clip from the game's own demo mode (Playwright + ffmpeg), regenerates the site, drives a real bot session through the real event pipe, adds a labeled synthetic platform cohort, computes the four bars, and writes the verdict back to `evidence.json` — so the next run's Signal is smarter.
